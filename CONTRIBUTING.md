# Contributing to V1-MultiAgent

首先，感谢你愿意为 V1-MultiAgent 做出贡献！🎉

## 如何贡献

### 提交文章

1. Fork 本仓库
2. 在 `articles/` 目录下添加你的文章
3. 文章使用 Markdown 格式
4. 提交 Pull Request

### 文章格式

```markdown
# 文章标题

> 作者：你的名字
> 日期：YYYY-MM-DD

## 概述

简要介绍文章内容

## 正文

...详细内容...

## 总结

...总结...

## 参考链接

- [链接1](url)
- [链接2](url)
```

### 报告问题

如果你发现文章中有错误或可以改进的地方：

1. 点击文章右上角的编辑按钮
2. 修改内容
3. 提交 Pull Request

或者创建一个 Issue，描述你发现的问题。

## 代码规范

- 使用清晰的中文表达
- 代码示例需要经过测试
- 配图请放在 `assets/images/` 目录
