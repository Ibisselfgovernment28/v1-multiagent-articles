- **Python not recognized:** Make sure Python is added to your system PATH during installation. Reinstall if necessary.

- **Missing dependencies:** Double-check you ran `pip install -r requirements.txt` in the project folder.

- **Error running scripts:** Confirm you are in the correct folder. Use the `cd` command carefully.

- **Permission issues:** Run the Command Prompt as an Administrator for some commands.
